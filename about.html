About this PWA

FitFuel is an installable, offline-capable web app built with Materialize CSS, Firebase Firestore, IndexedDB, and a custom service worker.

  • Installable (manifest + icons)
  • Offline fallback page
  • Online: saves to Firebase
  • Offline: saves to IndexedDB
  • Auto-sync when back online
